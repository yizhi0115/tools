欢迎使用个人工具库 (动态沙箱版)

为避免代码冲突（如 CSS 样式污染、全局 JS 变量覆盖），本框架已采用 Iframe 隔离架构与动态配置驱动模式。

标准接入流程：

  1. 将您以前保存的独立网页程序（例如 regex_tool.html）直接放入与本 index.html 同级的目录，或新建一个 tools 文件夹进行集中管理。无需对旧文件进行任何代码修改。
  2. 使用文本编辑器打开本 index.html，滚动到底部的 <script> 区域，找到 TOOL_CONFIG 数组。
  3. 在数组中新增一条对象配置，映射您的本地文件路径。示例：
    { id: 'regex-tool', title: '正则测试器', type: 'iframe', url: './regex_tool.html' }
  4. 保存文件并刷新页面，左侧导航栏将自动渲染新注册的工具。点击即可在右侧无缝加载并运行。
